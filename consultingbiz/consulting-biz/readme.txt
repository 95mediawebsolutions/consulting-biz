== Description ==

ConsultingBiz is a modern, professional and sophisticated free consultancy Wordpress Theme.

== Installation ==

1. From the admin panel, go to Appearance > Themes and click the Add New button.
2. Search for "Consulting Biz"
3. Install and activate

Yes! Consulting Biz is a free theme

= Do you offer support? =
Yes. We offer support on our website https://www.95media.co.uk/forums/forum/consulting-biz-wordpress-theme-support/
= Where can I find the theme options? =
All of our options can be found in the Customizer in 'Appearance > Customize'.

= Does this theme have any widget areas? =
Yes. consulting biz has 4 widget areas which you can add widgets to in Appearance > Widgets.
